## About Me

> **"코드는 단순한 논리의 집합이 아니라, 경험을 설계하는 도구입니다."**

저는 구조와 효율성, 그리고 사용자 경험의 흐름에 집착하는 **풀스택 개발자**입니다. 복잡한 기술적 과시보다는, 불필요한 요소를 덜어내고 서비스의 본질과 유용성을 드러내는 데 집중합니다.

소셜 미디어부터 공공 데이터 기반의 실시간 서비스까지, 다양한 플랫폼을 구축해온 경험을 바탕으로 구조적이면서도 유연한 개발을 지향합니다. 사용자가 원하기 전에 먼저 필요한 것을 해석하고 구현하는 것이 개발자의 역할이라 믿습니다.

현재 **한국**을 거점으로 활동하고 있으며, 새로운 프로젝트 제안은 언제나 환영합니다.

**실시간 지하철 정보 서비스**

<br>

### 02. DGD
**프리미엄 수하물 배송 서비스 (B2B)**
